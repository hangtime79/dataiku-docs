Stacking datasets
#################

.. contents::
	:local:


The "stack" recipe merges several datasets into one dataset. This recipe is the equivalent of a "union all" SQL statement.
The "stack" recipe can have pre-filters and post-filters. The filters documentation is available :doc:`here  <../other_recipes/sampling>`.

.. seealso::
    For more information, see the `Concept | Stack recipe <https://knowledge.dataiku.com/latest/data-preparation/visual-recipes/concept-stack-recipe.html>`_ article in the Knowledge Base.
