WARN_MISC_DISK_ROTATIONAL: Rotational hard drives
#################################################

The hard drive is a rotational hard drive, performance will be impacted. It is highly recommended to run DSS on SSD drives.

.. note:: Virtualization (such as RAID) may show a rotational hard drive while the underlying drive is a SSD.

Remediation
===========

Use SSD drives. Please refer to :ref:`this page<requirements.disks>` for more details.
