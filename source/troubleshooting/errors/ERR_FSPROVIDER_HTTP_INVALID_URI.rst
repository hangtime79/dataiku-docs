ERR_FSPROVIDER_HTTP_INVALID_URI: Invalid HTTP URI
##################################################

The provided URL is incorrect.

Please note that:

* The :doc:`HTTP Dataset </connecting/http>` only supports ``http://`` and ``https://`` URLs
* The :doc:`Download Recipe </other_recipes/download>` only supports ``http://``, ``https://`` and ``ftp://``
  URLs, other protocols must have a corresponding Connection


Remediation
===========

Fix the faulty URL.
