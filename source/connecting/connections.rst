.. _file-formats:

File formats
=========================

Here is a list of the file formats that DSS can read and write for files-based
connections (filesystem, HDFS, Amazon S3, SharePoint Online, HTTP, FTP, SSH).

Standard formats
-----------------

+--------------------------------------------------------+------+-----------------+
|                         Format                         | Read |      Write      |
+========================================================+======+=================+
| :doc:`Delimited values (CSV, TSV, ...) <formats/csv>`  | yes  | yes             |
+--------------------------------------------------------+------+-----------------+
| :doc:`formats/fixed-width`                             | yes  | no              |
+--------------------------------------------------------+------+-----------------+
| :doc:`Excel (XLS, XLSX, ...) <formats/excel>`          | yes  | only via export |
+--------------------------------------------------------+------+-----------------+
| :doc:`formats/avro`                                    | yes  | yes             |
+--------------------------------------------------------+------+-----------------+
| Custom format using                                    | yes  | no              |
| regular expression                                     |      |                 |
+--------------------------------------------------------+------+-----------------+
| :doc:`formats/xml`                                     | yes  | no              |
+--------------------------------------------------------+------+-----------------+
| :doc:`formats/json`                                    | yes  | no              |
+--------------------------------------------------------+------+-----------------+
| :doc:`formats/shapefile`                               | yes  | no              |
+--------------------------------------------------------+------+-----------------+
| MySQL Dump                                             | yes  | no              |
+--------------------------------------------------------+------+-----------------+
| Apache Combined log                                    | yes  | no              |
| format                                                 |      |                 |
+--------------------------------------------------------+------+-----------------+

Note that file-based formats can be read compressed: ZIP, GZIP, BZ2.
