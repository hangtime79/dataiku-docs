Neo4j
########

This plugin allows users to read and write from/to the Neo4j graph database.

It provides a connector to Neo4J graph databases. Thanks to this recipe you will be able to:

- export data from DSS to a Neo4J graph database
- import data from a Neo4J graph database to DSS
- run cypher queries in a Neo4J database using a macro

You can start using it by `downloading it <https://www.dataiku.com/product/plugins/neo4j/>`_.

.. note::

	**Not supported**: This plugin is :doc:`not a supported </troubleshooting/support-tiers>` feature.
