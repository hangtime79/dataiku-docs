Requirements
############

Dataiku version
^^^^^^^^^^^^^^^
- It is available for both Dataiku Cloud and Self-Managed.
- It is compatible with Dataiku 14+.

Python
^^^^^^
Python 3.9 & 3.10 are supported.

Infrastructure
^^^^^^^^^^^^^^

- The components of the plugin can run directly on DSS or in a container.
- The datasets containing the graph configurations can be hosted on any type of dataset.
- The files of the embedded graph databases can be hosted on any type of managed folder.
- **Visual Graph** uses `the embedded Kuzu graph database technology <https://kuzudb.github.io/docs/>`_.
