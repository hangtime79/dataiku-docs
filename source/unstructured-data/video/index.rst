Video
########

Video files can be manipulated with the help of the following plugin:

- `Plugin: Object detection in images <https://www.dataiku.com/dss/plugins/info/object-detection.html>`_: This plugin uses a deep learning model to detect the location and class of objects in video frames.

.. note::
	
	**Not supported**: This plugin is :doc:`Not supported </troubleshooting/support-tiers>`
