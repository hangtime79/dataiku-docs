AI Assistants
###############


.. note::

	AI Assistants should not be confused with the :doc:`LLM Mesh </generative-ai/index>`

	* The LLM Mesh is about empowering Dataiku users to leverage Generative AI for their use cases
	* AI Assistants are AI capabilities embedded within Dataiku itself to improve user productivity


Dataiku includes multiple AI Assistants designed to improve user productivity and help them discover the capabilities of the platform
