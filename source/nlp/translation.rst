Translation
############


.. contents::
	:local:


Translation is about translating text content from one language to another.

Dataiku provides several translation capabilities

Native translation
=========================

The native translation capability of Dataiku provides translation between `100 languages <https://huggingface.co/facebook/m2m100_418M/>`_ using open-source models.

It is an offline capability, meaning that it does not leverage a 3rd party API.


.. note::

	This capability is provided by the "Offline Translation" plugin, which you need to install. Please see :doc:`/plugins/installing`.

	This plugin is :doc:`Not supported </troubleshooting/support-tiers>`

Please see our `Offline Translation plugin page <https://www.dataiku.com/product/plugins/offline-translation/>`_ for detailed documentation.
