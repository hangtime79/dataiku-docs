Speech-to-Text
###############

.. contents::
	:local:


Speech to Text is the process of transforming audio files to text.

Dataiku provides several speech-to-text capabilities

Native speech-to-text
=========================

The native speech to text capability of Dataiku provides speech-to-text in English. It is an offline capability, meaning that it does not leverage a 3rd party API.

.. note::

	This capability is provided by the "Speech to Text" plugin, which you need to install. Please see :doc:`/plugins/installing`.

	This plugin is :doc:`Not supported </troubleshooting/support-tiers>`


Please see our `Speech to Text plugin page <https://www.dataiku.com/product/plugins/speech-to-text-cpu/>`_ for detailed documentation.
