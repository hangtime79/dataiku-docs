NLP using Deepl API
######################

The Deepl API integration provides translation between `28 languages <https://www.deepl.com/docs-api/translating-text/request/>`_
   

.. note::

	This capability is provided by the "Deepl Translation" plugin, which you need to install. Please see :doc:`/plugins/installing`.
