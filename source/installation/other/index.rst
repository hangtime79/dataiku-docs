Other installation options
###########################

In addition to the only supported ways to install DSS (:doc:`Dataiku Cloud Stacks for AWS </installation/cloudstacks-aws/index>`, :doc:`Dataiku Cloud Stacks for Azure </installation/cloudstacks-azure/index>`, :doc:`Dataiku Cloud Stacks for GCP </installation/cloudstacks-gcp/index>`, :doc:`Dataiku Cloud </installation/index>` and :doc:`Custom Dataiku installation on Linux </installation/custom/initial-install>`), DSS can be installed using other options.

.. warning::

	Dataiku does not provide support for any of these installation options.
