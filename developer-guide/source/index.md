# Dataiku - Developer Guide

Welcome to the Dataiku Developer Guide. This guide contains:

* Tutorials and example articles for developers and coder users of Dataiku.
* Reusable code samples for solving specific problems.
* Information on how to code in Dataiku, create applications, and operate Dataiku through its APIs.
* The Dataiku reference API documentation.


You might also find these other resources helpful:

- The {doc}`Reference Documentation<refdoc:index>` contains the primary documentation of Dataiku's concepts,
  interfaces, and features, information on how to install and configure Dataiku,
  and information for administrators on how to operate it.
- The {doc}`Knowledge Base<kb:index>` has various topics that can help you learn more about Dataiku,
  or find solutions to problems without having to ask for help.
  It mainly targets non-coders.
- The [Dataiku Academy](https://academy.dataiku.com) provides guided learning paths for you to follow,
  upskill, and gain certifications on Dataiku.
- The [Dataiku Community](https://community.dataiku.com) is where you can join the discussion, 
  get support, share best practices, and engage with other Dataiku users.
